Button 組建
---
可用參數

> text (string)

按鈕文字內容

> textColor (green, white, gray, black)

文字顏色

> textSize (string) (H1,H2,H3,H4,H5)

按鈕文字大小

> borderRadius (int)

邊框圓角

> border (int)

編框寬度

> borderColor (string)

邊框顏色

> fullWidth (boolean)

寬度最寬（受 parent 元素樣式引響）

> leftElement, rightElement (React Component)

可放在按鈕內文字左邊或者右邊的元素

> bold (boolean)

是否為粗體
